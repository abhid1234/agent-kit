# Team v0.2 Design Spec

## Overview

**Team** is the 4th core concept in agent-kit. It coordinates multiple agents on a single task using one of four strategies: sequential, parallel, debate, or hierarchical.

## API Surface

```typescript
import { Agent, Team } from '@avee1234/agent-kit';

const team = new Team({
  agents: [researcher, writer],
  strategy: 'sequential',
});
const result = await team.run('Research AI frameworks and write a summary');
```

### TeamConfig

| Option | Type | Required | Default |
|--------|------|----------|---------|
| `agents` | `Agent[]` | Yes | вҖ” |
| `strategy` | `'sequential' \| 'parallel' \| 'debate' \| 'hierarchical'` | Yes | вҖ” |
| `manager` | `Agent` | Only for hierarchical | вҖ” |
| `maxRounds` | `number` | No | `3` (debate only) |
| `maxDelegations` | `number` | No | `10` (hierarchical only) |

### TeamResult

```typescript
interface TeamResult {
  content: string;
  responses: AgentResponse[];
}

interface AgentResponse {
  agent: string;
  content: string;
  round?: number;
}
```

### Events

| Event | When | Data |
|-------|------|------|
| `team:start` | Team.run() called | `{ task, strategy, agents }` |
| `team:agent:start` | Before an agent runs | `{ agent, task, round? }` |
| `team:agent:end` | After an agent finishes | `{ agent, content, latencyMs }` |
| `team:round` | Debate round complete | `{ round, responses }` |
| `team:delegate` | Hierarchical delegation | `{ from, to, task }` |
| `team:end` | Team.run() complete | `{ content, totalLatencyMs }` |

## Strategies

### Sequential

```
Input вҶ’ Agent A вҶ’ Agent B вҶ’ Agent C вҶ’ Final Output
```

Each agent receives the original task + previous agent's output. Final agent's output is the result.

### Parallel

```
        в”ҢвҶ’ Agent A в”Җв”җ
Input в”Җв”Җв”јвҶ’ Agent B в”Җв”Җв”јвҶ’ Merge вҶ’ Final Output
        в””вҶ’ Agent C в”Җв”ҳ
```

All agents run concurrently via Promise.all. Outputs merged with agent attribution.

### Debate

```
Round 1: Agent A responds вҶ’ Agent B critiques
Round 2: Agent A revises  вҶ’ Agent B critiques
Round N: Agent A final answer
```

Agents alternate, each seeing all prior outputs. After maxRounds, first agent produces final response.

### Hierarchical

```
Manager вҶ’ delegate('researcher', 'find papers') вҶ’ result
Manager вҶ’ delegate('writer', 'summarize') вҶ’ result
Manager вҶ’ final answer
```

Manager gets auto-injected `delegate` tool with enum of agent names. Runs until it responds without delegating or hits maxDelegations cap.

## Success Criteria

1. All four strategies work with mock model
2. Team emits typed events throughout execution
3. Hierarchical manager can delegate multiple times before producing a final answer
4. Debate produces progressively refined output across rounds
5. Full test coverage on all strategies
